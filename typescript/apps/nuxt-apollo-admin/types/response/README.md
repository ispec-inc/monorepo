# Types/Response

## Overview
外部のAPIのResponseに関するtypeはこのディレクトリで定義する。  
このアプリケーションではgraphqlを利用するため、原則`query.gql`にてgraphqlのqueryと同時に定義する。  
typeとqueryは必ず同期させる。
(queryからのtypeの自動生成は要検討)
